# Актуальность темы исследования
В современных условиях цифровизации и роста требований к управлению персоналом ключевым фактором успеха организации становится эффективность руководящего состава. Удовлетворенность сотрудников своим руководителем напрямую влияет на мотивацию, продуктивность и текучесть кадров. Однако во многих компаниях оценка качества управления проводится эпизодически и без системного анализа, что затрудняет своевременное выявление проблем и принятие корректирующих мер
# Проблема:
Отсутствие системного инструмента мониторинга и анализа удовлетворенности руководителем. Существующие методы оценки (разовые опросы, анкетирование) часто не обеспечивают регулярность, наглядность и  оперативность реагирования, это приводит к задержкам в принятии решений, росту недовольства в коллективе, снижению эффективности управления 
# Цели и задачи
Создание централизованной системы отчетности для руководства ЦМ, которая обеспечит своевременное получение актуальной аналитики по выбранным участкам.
# Образ результата
Дашборд для оценки руководителей среднего звена – это автоматизированный инструмент, который: собирает данные через Яндекс Формы, визуализирует результаты в виде графиков, рейтингов и аналитических сводок, исключает ручную обработку данных.
